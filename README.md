# Sensor-Based Biomedical IoT Simulation

## Project
Smart Biomedical Cold Chain Temperature Monitoring and Alert System using Arduino

## Objective
To simulate a biomedical IoT system that monitors storage temperature and provides alerts when temperature exceeds the safe limit for temperature-sensitive healthcare products.

## Components
- Arduino UNO
- TMP36 Temperature Sensor
- 16×2 LCD Display
- PIR Motion Sensor
- Red LED
- Buzzer
- Potentiometer
- Breadboard

## Working
The Arduino reads temperature values from the TMP36 sensor and displays the real-time temperature on the LCD. When the temperature exceeds the predefined safety threshold, the system activates a red LED and buzzer alert. The PIR sensor is used for access monitoring of the storage unit.

## Software Used
- Tinkercad
- Arduino IDE

## Biomedical Applications
- Vaccine cold chain monitoring
- Medicine storage monitoring
- Blood sample preservation
- Laboratory sample monitoring

## Output
- Safe Temperature: LCD displays normal status
- High Temperature: Red LED + Buzzer alert
- Motion Detection: Access alert indication
